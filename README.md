# AI Chrome Extension

A Chrome Extension that provides AI-powered autocomplete for text fields on any website. Users receive text suggestions while typing and can accept them using the **Tab** key.

## Features

- AI text autocomplete
- Works on text inputs and textareas
- Accept suggestions using the Tab key
- Fast API responses
- Chrome Extension Manifest V3

## Technologies

- JavaScript
- HTML
- CSS
- Chrome Extension API
- Groq API

## Installation

1. Clone the repository.
2. Open `chrome://extensions/`.
3. Enable **Developer Mode**.
4. Click **Load unpacked**.
5. Select the project folder.
